Redact Extension Test Page

Instructions:

  1. Install the Redact Extension in Firefox
  2. Click the extension icon and add some sensitive words (like "password", "email", "ssn")
  3. Enable redaction
  4. Refresh this page to see the words replaced with <redacted>
